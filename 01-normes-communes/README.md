# Normes Communes — Éléments transversaux

Ce dossier contient les standards transversaux qui s'appliquent à tous les objets métier du Work Structure Dictionary.

## Contenu

- [Statuts](statuts.md) — Cycles de vie et statuts des objets
- [Priorités](priorites.md) — Niveaux de priorité
- [Rôles](roles.md) — Définition des rôles communs
- [Catégories](categories.md) — Catégories des objets
- [Liens croisés](liens-croises.md) — Liens entre objets

## Rôle

Ces normes sont le **tronc "Norme"** du projet. Elles définissent les éléments transversaux communs à tous les objets métier :

- **Statuts** : New, In Progress, Closed, etc.
- **Priorités** : Low, Medium, High, Very High
- **Rôles** : Définis une fois, réutilisés partout
- **Catégories** : Classification transversale
- **Liens croisés** : Relations entre objets

Chaque objet métier (OBJ-001 à OBJ-041) hérite de ces normes. Un objet peut en étendre les définitions mais ne peut les contredire.
