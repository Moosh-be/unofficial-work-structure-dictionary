# Livrables (OBJ-002)

Les livrables représentent l'ensemble des livrables tangibles produits par le projet.

Ils constituent les résultats attendus, vérifiables et acceptés par les parties prenantes.

---

## Rôle

Les livrables permettent de :

- définir clairement **ce qui est produit**
- servir de **base de vérification** de l'avancement
- faciliter la **réception et l'acceptation**
- constituer une **trace documentée** des réalisations

---

## Classification

Les livrables peuvent être classés selon leur nature :

- **Livrables principaux** : Résultats finaux du projet
- **Livrables intermédiaires** : Jalons de production
- **Livrables de gestion** : Documents de pilotage (plans, rapports)
- **Livrables transversaux** : Documentation, formations, supports

---

## Principes

Un livrable doit être :

- **Spécifié** clairement dès sa conception
- **Vérifiable** par des critères objectifs
- **Accepté** formellement par son commanditaire
- **Traçable** depuis sa demande initiale jusqu'à sa livraison
