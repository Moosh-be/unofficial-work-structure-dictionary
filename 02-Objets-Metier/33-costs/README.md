# Costs (OBJ-033)

Les Costs (dépenses) enregistrent les coûts directs du projet.

---

## Rôle

Les Costs permettent de :

- **suivre** les dépenses effectives
- **comparer** estimé vs réel
- **justifier** les postes de coûts
- **facturer** les prestations

---

## Types de coûts

| Type | Description |
|------|-------------|
| **Personnel** | Coût des heures travaillées |
| **Matériel** | Achats d'équipements |
| **Licences** | Coût des logiciels |
| **Sous-traitance** | Prestations externes |
| **Déplacement** | Frais de transport |
| **Formation** | Coûts de formation |
| **Autre** | Autres dépenses |

---

## Suivi budgétaire

Les Costs s'intègrent au budget projet :

- **Budget prévisionnel** : Estimé par WorkPackage ou nœud WBS
- **Engagements** : Contrats signés, commandes passées
- **Réalisations** : Dépenses facturées et payées
- **Reste à faire** : Prévisionnel restant

---

## Réconciliation

- **Monthly reconciliation** : Comparaison budget vs réalisé
- **Variance analysis** : Écarts et causes
- **Forecasting** : Prévisions de clôture

---

## Principe

> Un coût non tracé est un budget non contrôlé.
> Un budget non mis à jour est un budget faux.
