# Risks (OBJ-008)

Les risques représentent les événements ou conditions incertaines qui, s'ils survenaient, auraient un effet (positif ou négatif) sur les objectifs du projet.

---

## Rôle

Les risques permettent de :

- **anticiper** les problèmes potentiels
- **préparer** des plans de mitigation
- **exploiter** les opportunités
- **piloter** de façon proactive

---

## Classification

Les risques peuvent être classés selon :

- **Nature** : Technique, Organisationnelle, Financière, Externe, Humaine
- **Impact** : Majeur, Moyen, Mineur
- **Probabilité** : élevée, moyenne, faible

---

## Processus de gestion

1. **Identification** : Lister les risques potentiels
2. **Analyse** : Évaluer probabilité et impact
3. **Planification** : Définir les réponses (éviter, atténuer, transférer, accepter)
4. **Suivi** : Monitorer les risques identifiés et détecter les nouveaux

---

## Principe

> Un risque identifié et géré est un problème évité.
> Un risque non identifié est une surprise.
