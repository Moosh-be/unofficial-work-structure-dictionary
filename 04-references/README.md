# References — Index croisés

Ce dossier indexe les outils, méthodes et normes internationales en relation avec le Work Structure Dictionary.

## Pourquoi ?

Pour :
- Savoir quel outil couvre quels objets de la norme
- Savoir quelle méthode utilise quels artefacts de gestion
- Savoir quelle norme internationale recouvre quel objet

## Contenu

- [Outils](outils.md) — Jira, MS Project, Planner, Excel, Confluence, Notion, Obsidian, etc.
- [Méthodes](methodes.md) — SAFe, PMBOK, PM², Lean, PRINCE2, ITIL 4, TOGAF, etc.
- [Normes](normes.md) — ISO 21502, ISO 9001, COBIT, etc.

## Comment contribuer ?

Chaque entrée doit contenir :
- Nom de l'outil/méthode/norme
- Lien vers documentation officielle
- Objets de la norme couverts
- Comment synchroniser (si applicable)
