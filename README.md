# Work Structure Dictionary

Un standard de structure de données générique pour structurer les artefacts de gouvernance d'unités de travail (projets, programmes, produits, chantiers).

Ce dépôt propose des conventions pour rendre les artefacts de gestion (tâches, rôles, décisions, dates, budgets, ressources, exigences, etc.) plus **interopérables**, plus **durables**, plus **réutilisables**, et moins captifs des outils et méthodes spécifiques.

**Pas une méthode. Pas un framework. Un langage commun.**
