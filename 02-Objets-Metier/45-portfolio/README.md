# Portfolio (OBJ-045)

Un Portfolio est un conteneur de niveau supérieur qui regroupe plusieurs projets pour une vue d'ensemble stratégique.

---

## Rôle

Le Portfolio permet de :

- **regrouper** les projets par portefeuille, axe stratégique, domaine fonctionnel
- **consolider** les indicateurs : budget total, nombre de projets, statut global
- **tracer** la hiérarchie de portefeuilles (portefeuilles parents/enfants)
- **faciliter** le reporting directionnel sans plonger dans les détails opérationnels

---

## Classification

Un Portfolio peut être classé selon :

- **Type** : Stratégique, Opérationnel, Transverse, Temporaire
- **Périmètre** : Organisationnel, Programme, Domaine, Thème

---

## Principe

> Un portfolio est une question de regroupement, pas de niveau de détail.
