# News (OBJ-029)

Les News (actualités) informent les membres de l'équipe et parties prenantes des événements importants du projet.

---

## Rôle

Les News permettent de :

- **communiquer** les informations officielles
- **alerter** sur les événements importants
- **centraliser** les annonces
- **maintenir** l'équipe informée

---

## Types de News

- **Annonces officielles** : Décisions, changements de gouvernance
- **Mises à jour de version** : Releases, mises en production
- **Alertes** : Incidents critiques, blocages
- **Événements** : Réunions, formations, deadlines
- **Célébrations** : Jalons atteints, succès

---

## Fonctionnalités

Une News contient :

- Un **titre** accrocheur
- Un **corps** de message (formaté)
- Un **auteur** (qui publie)
- Une **date** de publication
- Des **catégories** pour le filtrage
- La possibilité d'**épingler** les annonces importantes

---

## Bonnes pratiques

- **Cibler** : Adapter le message aux destinataires
- **Simplifier** : Aller à l'essentiel
- **Ponctuer** : Annoncer les résultats (pas seulement les problèmes)
- **Archiver** : Consulter l'historique des communications

---

## Principe

> Une communication fluide est un projet qui avance.
> Une News non lue est une information perdue.
