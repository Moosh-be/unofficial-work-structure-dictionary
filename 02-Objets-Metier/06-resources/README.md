# Resources (OBJ-006)

Les ressources représentent l'ensemble des moyens alloués au projet : humains, matériels, logiciels, financiers.

Cet objet couvre les ressources en tant que capacité disponible, distincte de leur affectation concrète.

---

## Rôle

Les ressources permettent de :

- inventorier les **moyens disponibles**
- planifier l'allocation des **capacités**
- suivre la **consommation** des ressources
- anticiper les **besoins complémentaires**

---

## Types de ressources

- **Humaines** : Compétences, disponibilités
- **Matérielles** : Équipements, infrastructures
- **Logicielles** : Licences, outils
- **Financières** : Budgets, enveloppes

---

## Distinction importante

> La ressource est la capacité disponible.
> L'assignation est l'affectation effective.

Cette séparation permet de gérer la disponibilité indépendamment de l'usage.
