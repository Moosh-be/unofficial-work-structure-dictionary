# Availability (OBJ-047)

Un disponibilité représente une période pendant laquelle une ressource n'est pas disponible pour travailler (congés, formation, indisponibilité).

---

## Rôle

L'Availability permet de :

- **tracer** les périodes d'indisponibilité d'une ressource (congés, formation, maladie, etc.)
- **informer** le capacity planning en distinguant indisponibilité programmée de surallocation réelle
- **documenter** les absences pour le reporting RH et project management

---

## Classification

Les types d'indisponibilité :

- **Congés payés** — Vacances, temps libre
- **RTT** — Heures supplémentaires récupérées
- **Formation** — Période de formation certifiante ou interne
- **Maladie** — Arrêt de travail (médical)
- **Maternité/Paternité** — Congé parental
- **Service militaire** — Appel sous les drapeaux
- **Autre** — Indisponibilité non classée

---

## Principe

> Une indisponibilité planifiée n'est pas un conflit de capacity — c'est un paramètre du planning.
