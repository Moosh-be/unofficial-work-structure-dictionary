# Budgets (OBJ-009)

Les budgets représentent les enveloppes financières allouées au projet et à ses différentes composantes.

---

## Rôle

Les budgets permettent de :

- **planifier** les dépenses prévisionnelles
- **suivre** la consommation effective
- **anticiper** les dépassements
- **justifier** la maîtrise financière

---

## Structure budgétaire

Un budget peut être décomposé en :

- **Investissements initiaux** (CAPEX) : Équipements, licences, mise en place
- **Fonctionnement** (OPEX) : Coûts récurrents (personnel, maintenance, infrastructure)
- **Contingences** : Réserve pour risques identifiés
- **Réserves de management** : Réserve pour risques inconnus

---

## Principe

> Un budget n'est pas une contrainte rigide, mais un outil de pilotage.

L'important est la **transparence** sur la consommation et les écarts.
